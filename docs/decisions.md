# Engineering Decisions

This document records the most important engineering decisions made during the development of the Garden Management Platform.

---

# Decision 001

**Date**

2026-06-29

**Topic**

Primary Server Platform

**Decision**

The Raspberry Pi 5 will be used as the primary server of the platform.

**Reason**

* Low power consumption
* Silent operation
* Runs continuously
* Excellent Linux support
* Large community
* Suitable performance for the planned infrastructure

**Alternatives Considered**

* Mini PC
* VPS only
* Repurposed desktop computer

**Status**

✅ Accepted
